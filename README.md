<h3 id="acerca-de-unalbot">📄 Acerca de UNalBot🤖</h3>
<p><strong>UNalBot</strong> es un chatbot diseñado y creado por un estudiante de la Universidad Nacional de Colombia para servir como un asistente virtual dentro de la misma universidad. Esta aplicación utiliza técnicas de procesamiento de lenguaje natural para interpretar y responder a las consultas en lenguaje humano de manera eficiente y efectiva.</p>

<h3 id="cómo-funciona">⚙️ Cómo Funciona</h3>
<p>El bot se entrena con un conjunto de datos en formato JSON, que incluye varios intents y sus correspondientes patrones y respuestas. Utilizando NLTK para el procesamiento del texto y Keras/TensorFlow para el modelo de aprendizaje automático, UNalBot puede generar respuestas adecuadas basadas en las consultas recibidas.</p>
